01 / MOBILE REMOTE CONTROL

我不坐在电脑前，
也能继续推进任务。

重点不是“手机上写代码”，而是手机可以远程控制那台正在运行 Codex 的电脑。

手机负责看进度、补指令、批准下一步；代码、凭证、权限还是留在主机上。

02 / CHROME EXTENSION

让 Codex 用我的登录态处理网页任务

后台、邮箱、CRM、公众号后台这类任务，以前经常卡在登录态。Chrome 插件让 Codex 在授权后读取和操作网页。

@Chrome 打开我已经登录的后台，只读检查最近一篇草稿的标题、封面和正文是否一致。不要发布，不要修改任何内容。
