# Motion Effects Reference

Use this file when proposing effects to a user or selecting motion for a scene. Describe effects in plain viewing language first, then implement with Remotion or HyperFrames.

## Core Effects

| Effect | User-facing explanation | Best for | Avoid when |
| --- | --- | --- | --- |
| Icon orbit | 图标从四周飞入，形成双层旋转星云，再收束到标题或总览。 | tool ecosystems, plugin intros, partner/logo walls | only 3-5 icons are available |
| Staggered card grid | 卡片按节奏依次弹入，最后停成一张完整清单。 | TOP lists, feature lists, comparison boards | copy is too long for cards |
| Light sweep | 蓝白光扫过画面，把一幕自然带到下一幕。 | clean tech transitions | dark or documentary scenes |
| Page turn | 单个卡片像翻书一样转到正面，进入详情页。 | one feature/plugin/product per page | dense multi-item overviews |
| Spotlight zoom | 镜头推近截图局部，给关键按钮或数据加光圈。 | screenshot tutorials and UI demos | source screenshot is low-res |
| Typewriter | 文案按输入节奏出现，模拟提示词、代码、搜索或生成过程。 | prompts, code, search, AI generation | ordinary explanatory paragraphs |
| Counter reveal | 数字或指标从低到高滚动，再定格成结果。 | data explainer, performance, ad proof | no credible data source |
| Parallax photo cards | 图片分层轻微移动，制造空间感。 | travel, culture, brand mood films | exact UI inspection is needed |

## Rhythm Rules

- Give dense cards a readable hold after entrance.
- Keep the strongest effect to one or two moments per short video.
- Use motion to clarify grouping, priority, cause-effect, or transformation.
- Do not let particles, trails, or glow cover the message.
- Make transitions feel motivated: orbit into title, screenshot into proof, card grid into summary.

## Example Effect Packages

- **Tech opener**: icon orbit + light sweep + title split reveal + overview grid.
- **Vertical TOP list**: title pop + staggered cards + focus beats + final full-list hold.
- **Product explainer**: screenshot spotlight + typewriter prompt + preview expansion + capability cards.
- **Brand video**: logo orbit + palette sweep + application mockups + soft parallax.
- **Travel promo**: map trace + photo cards + itinerary timeline + cinematic location title.
